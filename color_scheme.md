# Brand Color Scheme

This document outlines the core color scheme, providing a structured palette for consistent application across various mediums including digital interfaces, hardware, printed materials, and merchandise.

The theme utilizes a warm cool green-gray neutral scale combined with a vibrant sage green accent.

## Neutral Palette (Warm Cool Green-Gray)

These colors form the foundational base, suitable for backgrounds, large surfaces, and structural elements.

| Role | Hex Code | Suggested Usage |
| :--- | :--- | :--- |
| **Base Level 1** | `#f2f4f3` | Primary backgrounds, exterior surfaces |
| **Base Level 2** | `#ffffff` | Elevated surfaces, primary focal areas |
| **Base Level 3** | `#f8faf9` | Interactive states, subtle highlights |
| **Secondary Base** | `#f0f3f1` | Supporting areas, secondary structural elements |
| **Secondary Base 2**| `#e8ede9` | Alternate supporting areas |
| **Subtle Divider** | `#dde3df` | Light borders, subtle separation lines |
| **Soft Divider** | `#e8ede9` | Faint inner dividing lines |
| **Strong Divider** | `#c8d0cb` | Distinct borders, prominent structural outlines |

## Ink & Typography Palette

| Role | Hex Code | Suggested Usage |
| :--- | :--- | :--- |
| **Primary Ink** | `#1a2420` | High-contrast main text, primary logos |
| **Secondary Ink** | `#4a5e55` | Supporting text, secondary labels |
| **Muted Ink** | `#6e8078` | Tertiary text, subtle graphical elements |
| **Faint Ink** | `#96a89e` | Placeholders, decorative inactive elements |

## Brand Accent: Sage Green

The primary accent color used for brand highlights, interactive elements, and focal points.

| Role | Hex Code / Value | Suggested Usage |
| :--- | :--- | :--- |
| **Primary Accent** | `#3d7a5a` | Core brand color, primary actions, key highlights |
| **Accent Dark** | `#336650` | Active states, deepened brand elements |
| **Accent Darkest** | `#2a5542` | High-emphasis interactive states |
| **Accent Light** | `#eaf2ec` | Soft brand washes, subtle selections |
| **Accent Lighter** | `#daeade` | Secondary brand washes |
| **Accent Border** | `#b8d6c0` | Outlines and borders for branded elements |
| **Accent Wash** | `rgba(61,122,90,0.16)` | Soft glows, shadows, or transparent overlays |

## High-Contrast / Dark Palette

A darker variant of the theme designed to provide striking contrast against the lighter neutral palette.

| Role | Hex Code / Value | Suggested Usage |
| :--- | :--- | :--- |
| **Dark Base** | `#1e2d27` | High-contrast backgrounds, structural framing |
| **Dark Ink** | `#a8c4b4` | Text or elements on dark backgrounds |
| **Dark Ink Bright** | `#ffffff` | High-emphasis elements on dark backgrounds |
| **Dark Accent** | `rgba(61,122,90,0.35)` | Active highlights on dark backgrounds |
| **Dark Highlight** | `rgba(255,255,255,0.07)`| Subtle interactions on dark backgrounds |
